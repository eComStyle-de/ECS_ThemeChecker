eComStyle.de_ThemeChecker
==========================


Installation: 

    1. Copy content of folder "copy_this" into /  (=> rootfolder)
	
	2. Delete content of /tmp
	
	3. Activate in "Extensions" -> "Modules"

	4. Be Happy :)
	
Shopversion:

	OXID eShop CE 4.6.0 - 4.7.4
